# 📚 Nejinho Public

Sistema educativo enfocado en la **Educación de Jóvenes y Adultos (EJA)**, desarrollado como parte de un proyecto académico.

Este proyecto busca facilitar la **interacción entre docentes y estudiantes**, permitiendo la creación, gestión y resolución de actividades educativas de forma digital.

---

## 🚀 Demo

La aplicación ya se encuentra en producción:

🔗 https://nejinho.herokuapp.com/

> ⚠️ Nota: Para registrarse, se requiere una clave de acceso disponible en el controlador de login.

---

## 🎯 Objetivo

Desarrollar herramientas digitales que apoyen el proceso de enseñanza-aprendizaje en programas de alfabetización de jóvenes y adultos, especialmente en contextos donde el acceso a educación presencial es limitado.

---

## 🧩 Características

### 🖥️ Sistema Web (Docentes)
- Registro de usuarios mediante clave de acceso
- Creación y gestión de actividades
- Edición y control de disponibilidad de contenido
- Visualización de actividades y autores
- Consulta de respuestas de estudiantes

### 📱 Aplicación Móvil (Estudiantes)
- Visualización de actividades disponibles
- Envío de respuestas
- Registro de nombre (firma) en actividades
- Interfaz simple e intuitiva

---

## 🛠️ Tecnologías utilizadas

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** C#  
- **Arquitectura:** Cliente - Servidor  

---

## 🧠 Contexto del Proyecto

Este sistema surge como respuesta a los desafíos de la educación durante la pandemia de COVID-19, donde la enseñanza presencial se vio interrumpida.

La plataforma busca ofrecer una alternativa accesible para estudiantes de EJA, quienes frecuentemente enfrentan barreras tecnológicas, económicas y sociales.
